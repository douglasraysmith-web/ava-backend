# AVA Account-Side Blockers v7.8.4

These are not source-code blockers. They require access to the owner's real accounts.

## Required
- GitHub repo selected or created.
- Railway project/service created.
- Railway Postgres attached.
- Railway environment variables set.
- Database migration run.
- Netlify/mtthorne.com route updated for `/AV/AVA`.
- Real backend URL inserted into `ava-config.js` or server-side config.
- Owner token generated and stored privately.
- Live proof scripts run against real URLs.

## Optional for first owner demo
- OpenAI/model provider key.
- R2/S3 object storage.
- E2B/container sandbox.
- Voice provider.
- Final visible-being artwork.
