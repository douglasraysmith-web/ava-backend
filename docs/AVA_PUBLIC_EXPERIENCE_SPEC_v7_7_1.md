# Ava Public Experience Spec v7.7.1

Status: owner-pilot / public-safe preview. Not a verified live deployment.

## Public site sections

- Ask Ava: text-first chat entry, voice-ready controls, AI disclosure, useful example prompts.
- Plan a Theater: room size, screen, projector/TV, seating, speaker layout, wiring, acoustic basics, and budget path.
- Fix a Problem: guided troubleshooting for no sound, HDMI handshake, eARC, weak bass, projector image, remote/control issues.
- Find Drivers and Sources: public-safe explanation of driver/protocol lookup without exposing private or dealer-only files.
- Request Help: owner-review path without payment/account/customer-data claims.
- AV Knowledge Base: readable public answers and FAQ summaries kept separate from private owner materials.

## Answer shape

1. Direct answer.
2. Why it matters.
3. Safest next step.
4. Optional owner/business CTA when useful.

## Boundary

Ava must not claim public deployment, customer-data readiness, payment/account readiness, hardware control, compatibility, code compliance, driver availability, command safety, or field readiness unless built, enabled, and verified.
